# ADR-002 — Périmètre V1

## Statut
Adopté

## Décision
La V1 sert uniquement à recruter et suivre des talents immobiliers.

## Inclus
- personnes ;
- organisations ;
- relations ;
- pipeline ;
- import CSV ;
- Brevo ;
- relances ;
- événements de recrutement.

## Exclu
- gestion locative ;
- formation ;
- transaction ;
- marketplace ;
- knowledge graph avancé ;
- application mobile native.
