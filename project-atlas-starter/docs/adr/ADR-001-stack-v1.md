# ADR-001 — Stack V1

## Statut
Adopté

## Décision
Utiliser Next.js, Supabase/PostgreSQL, n8n, Brevo, OpenAI et Vercel.

## Contexte
Budget initial de 10 000 €, nécessité d'un produit rentable rapidement, ambition de créer une société à forte valeur.

## Conséquences
- lancement rapide ;
- coûts initiaux limités ;
- base PostgreSQL pérenne ;
- dépendance Supabase maîtrisable ;
- possibilité d'extraire des services plus tard.
