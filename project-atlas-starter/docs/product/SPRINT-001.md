# SPRINT-001 — Core Foundation

## Objectif
Obtenir une première application accessible en ligne avec authentification et gestion des personnes, organisations et relations.

## Durée cible
7 à 10 jours ouvrés.

## Livrables
- dépôt GitHub opérationnel ;
- projet Next.js déployé ;
- projet Supabase configuré ;
- authentification ;
- tables `people`, `organizations`, `relationships` ;
- liste des personnes ;
- création et modification d'une personne ;
- import CSV simple ;
- journal minimal des changements.

## Definition of Done
- un utilisateur peut se connecter ;
- créer une personne ;
- modifier une personne ;
- rattacher une organisation ;
- créer une relation de recrutement ;
- importer un CSV ;
- retrouver un contact par recherche ;
- aucune donnée de test sensible en production.
