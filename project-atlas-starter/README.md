# Project Atlas

Nom de code temporaire du futur éditeur logiciel.

## Objectif V1
Créer un produit rentable permettant d'identifier, qualifier, contacter et suivre des talents immobiliers, puis d'étendre le socle à d'autres métiers.

## Stack figée pour la V1
- Frontend et API : Next.js
- Base, authentification et stockage : Supabase / PostgreSQL
- Automatisations : n8n
- Emails et SMS : Brevo
- IA : OpenAI via une couche d'abstraction
- Hébergement frontend : Vercel
- Documentation et suivi : GitHub
