# ADR-001 - Stack V1

## Statut

Adopte

## Decision

Utiliser Next.js, Supabase/PostgreSQL, n8n, Brevo, OpenAI et Vercel.

## Contexte

Budget initial de 10 000 EUR, necessite d'un produit rentable rapidement, ambition de creer une societe a forte valeur.

## Consequences

- lancement rapide ;
- couts initiaux limites ;
- base PostgreSQL perenne ;
- dependance Supabase maitrisable ;
- possibilite d'extraire des services plus tard.
